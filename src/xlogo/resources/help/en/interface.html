Interface Guide

The Main Window

  • Menu Bar - File, Edit, Tools, Options, and Help menus
  • Command Line - Enter Logo instructions here and press Enter to execute
  • Drawing Area - The canvas where the turtle draws
  • Tool Bar - Zoom, edit tools, and the Play button
  • Command History - Shows previous commands and responses
  • Stop Button - Interrupts program execution
  • Editor Button - Opens the procedure editor

The Procedure Editor

The editor lets you write and save procedures (reusable code). Open it by:

  • Typing ed in the command line
  • Clicking the Editor button
  • Using the keyboard shortcut Alt+E

Main Command

At the bottom of the editor, you can define a "main command" - the instruction that runs when you click the Play button in the main window. This command is saved with your .lgo file.

File Menu

  • New - Create a new workspace (clears all procedures and variables)
  • Open - Open a saved .lgo file
  • Save / Save As - Save your procedures
  • Capture Image - Save the drawing area as an image
  • Print - Print the current drawing
  • Quit - Exit XLogo
