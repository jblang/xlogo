Primitives Reference

The turtle is controlled by internal commands called primitives.

Movement

forward, fd n
Move the turtle forward n steps. Example: fd 100
back, bk n
Move the turtle backward n steps. Example: bk 50
right, rt n
Turn the turtle n degrees to the right. Example: rt 90
left, lt n
Turn the turtle n degrees to the left. Example: lt 45
home
Return the turtle to position [0 0] facing up (heading 0).
setpos, setposition [x y]
Move the turtle to coordinates [x y]. Example: setpos [100 -50]
setx x
Move the turtle horizontally to x. Example: setx 100
sety y
Move the turtle vertically to y. Example: sety -50
setxy x y
Same as setpos [x y]. Example: setxy 100 -50
setheading, seth n
Set the turtle's heading to n degrees (0 = up, like a compass). Example: seth 90

Drawing Shapes

circle radius
Draw a circle with the given radius around the turtle. Example: circle 50
arc radius angle1 angle2
Draw an arc with given radius between angle1 and angle2. Example: arc 50 0 90
dot [x y]
Draw a dot at coordinates [x y]. Example: dot [100 100]
label text
Write text at the turtle's position. Example: label [Hello World]

Pen Control

pendown, pd
Lower the pen (turtle will draw when moving).
penup, pu
Raise the pen (turtle moves without drawing).
penerase, pe
Pen erases marks it passes over.
penpaint, ppt
Return to normal drawing mode.
setpenwidth, setpw n
Set pen thickness in pixels. Example: setpw 5
penwidth, pw
Returns current pen thickness.
setpenshape, setps 0|1
Set pen shape: 0 = square, 1 = round.

Colors

setpencolor, setpc color
Set the pen color. Use a color name or [r g b] list.
Examples: setpc "red or setpc [255 0 0]
setscreencolor, setsc color
Set the background color. Example: setsc "blue
pencolor, pc
Returns current pen color as [r g b].
screencolor, sc
Returns current screen color as [r g b].
findcolor, fc [x y]
Returns the color of the pixel at [x y].

Turtle Visibility

showturtle, st
Make the turtle visible.
hideturtle, ht
Hide the turtle (drawing is faster when hidden).
setshape n
Set turtle shape (0-6). 0 is triangular.
shape
Returns current turtle shape number.

Screen Control

clearscreen, cs
Clear the drawing area and return turtle home.
wash
Clear the drawing area but keep turtle in place.
resetall
Reset all settings to defaults and clear screen.
window
Turtle can travel outside the drawing area.
wrap
Turtle wraps around to opposite side when leaving.
fence
Turtle is confined to the drawing area (error if it tries to leave).
perspective
Enable 3D mode. Use window/wrap/fence to exit.
setscreensize [w h]
Set screen dimensions. Example: setscreensize [800 600]
screensize
Returns current screen size as [w h].

Queries

pos, position
Returns turtle position as [x y]. Example: print pos
x
Returns the turtle's x-coordinate.
y
Returns the turtle's y-coordinate.
z
Returns the turtle's z-coordinate (3D mode only).
heading
Returns the turtle's current heading.
towards [x y]
Returns heading needed to face point [x y].
distance [x y]
Returns distance from turtle to point [x y].

Control Structures

repeat n [commands]
Repeat commands n times. Example: repeat 4 [fd 100 rt 90]
if condition [commands]
Execute commands if condition is true. Example: if :x > 10 [print "big]
ifelse condition [true-cmds] [false-cmds]
Execute first list if true, second if false.
for [var start end step] [commands]
Loop with counter. Example: for [i 1 10 1] [print :i]
while [condition] [commands]
Repeat while condition is true.
stop
Exit the current procedure.
output, op value
Return a value from a procedure. Example: output :x + 1

Grid

grid width height
Draw a grid with given cell dimensions. Example: grid 50 50
stopgrid
Remove the grid.
setgridcolor color
Set the grid color. Example: setgridcolor "gray
