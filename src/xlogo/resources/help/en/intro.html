Introduction to XLogo

Logo is a programming language developed in the 1960s by Seymour Papert. Papert developed an influential theory on learning called "constructionism" which could be summarized as "learning by doing."

Logo is an excellent language for developing mathematical and logic skills. It teaches the basics of programming concepts like loops, conditionals, and procedures. Users control an object called a turtle that moves around the screen using simple commands like forward, back, right, and left. As it moves, the turtle leaves a trail, allowing you to create drawings.

What is XLogo?

XLogo is a Logo interpreter - your instructions are executed immediately, and you can see the results on screen right away. This intuitive graphical approach makes Logo ideal for beginners, especially children!

Features

  • Supports multiple languages (Arabic, Asturian, Catalan, English, Esperanto, French, Galician, German, Greek, Hungarian, Italian, Portuguese, and Spanish)
  • 2D and 3D graphics capabilities
  • Sound and animation support
  • Network programming features
  • Cross-platform (Windows, macOS, Linux)
