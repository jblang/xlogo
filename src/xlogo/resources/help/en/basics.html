Getting Started

To move the turtle, use predefined commands called primitives. This chapter covers the basic primitives for controlling the turtle.

Drawing Shapes

Square

Move forward, turn 90 degrees, repeat 4 times:

repeat 4 [fd 100 rt 90]

Triangle

For an equilateral triangle, the external angle is 180 - 60 = 120 degrees:

repeat 3 [fd 100 rt 120]

Any Regular Polygon

For a polygon with n sides, the turning angle is 360/n:

; Pentagon (5 sides): 360/5 = 72
repeat 5 [fd 80 rt 72]

; Hexagon (6 sides): 360/6 = 60
repeat 6 [fd 60 rt 60]

; Circle approximation (360 sides)
repeat 360 [fd 2 rt 1]

Creating Procedures

Save commands as reusable procedures. Open the editor and write:

to square
  repeat 4 [fd 100 rt 90]
end

Click the turtle button to save. Now type square to draw a square!

Procedures with Parameters

Make flexible procedures using variables (prefixed with :):

to square :size
  repeat 4 [fd :size rt 90]
end

Now square 50 draws a small square, square 200 draws a large one.

Tip: Use penup (pu) to move without drawing, then pendown (pd) to resume drawing.

Try It!

Draw a house using these procedures:

to house
  square 100      ; main body
  fd 100          ; move to roof position
  triangle 100    ; roof
end

to triangle :size
  repeat 3 [fd :size rt 120]
end
