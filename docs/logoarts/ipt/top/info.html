Modes

XLogo can be switched between several screen, pen and setup modes.

Screen -switch between four different modes:

  1. Window (default)
  2. Wrap
  3. Fence
  4. 3D / Perspective

Pen -switch between three different modes:

  1. PenPaint (default)
  2. PenReverse
  3. PenErase

XLogo -enter or stop four different modes:

  1. Animation
  2. Axis (not available in 3D mode)
  3. Grid (not available in 3D mode)
  4. Trace
