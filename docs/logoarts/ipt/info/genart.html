Generative Art

- XLogo art

Generative art is visual art created by a computer, using a set of mathematical rules (an algorithm). Usually, a random process is incorporated to add variability, so each design is slightly different.

Can also be called algorithmic art, where artists are called algorists.

Here is a list of programs on this site that produce a series of random generative arts.
